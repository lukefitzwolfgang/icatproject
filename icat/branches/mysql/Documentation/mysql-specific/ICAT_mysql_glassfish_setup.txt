Glassfish config for MySQL (assuming a GF server with oracle ICAT already installed)

Install the drivers
- drop mysql-connector-java-5.1.14-bin.jar into glassfish\domains\domain1\lib\ext (gf will require a restart)

Set up the connection pool
- In the admin console (web pages) open up Resources -> JDBC -> Connection pools
- Select the icat3 pool
- Change the 'Datasource Classname' to 'com.mysql.jdbc.jdbc2.optional.MysqlDataSource'
- Click Save
- Open the Additional properties tab
- Set user to 'icat'
- Set password to the correct value
- Set url to 'jdbc:mysql://peaceful.esc.rl.ac.uk:3306/icatisis33_test'
- Click Save

Deploy the mqsql branck of ICAT (build icat3-exposed and deploy icat3-api-ws.jar
